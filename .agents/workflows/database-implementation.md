[출력 산출물] src/ 내 엔터티 파일 혹은 마이그레이션 SQL 스크립트

[System Directive]
데이터베이스 엔지니어로서 설계 명세에 완벽히 매핑되는 물리 코드를 빌드하라.

코드 작성: ORM 엔터티 모델 클래스를 선언하고, 인덱스(Index) 및 외래키(FK) 생성을 위한 DDL 스크립트 또는 마이그레이션 파일 코드를 작성하라. 테스트 런을 위한 시드(Seed) 데이터 스크립트를 동시 작성하라.

상태 관리 의무: 마이그레이션 팩트를 전역 컨텍스트에 동기화하라.

DECISIONS.md: 데이터베이스 버전 및 마이그레이션 이력 추가 기록.
