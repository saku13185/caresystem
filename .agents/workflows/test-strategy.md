QA 엔지니어로서 시스템 안정성을 계측하기 위한 하네스 엔지니어링 검증 코드를 설계하라.

5대 검증 항목 반영: 데이터 무결성 검증, AttentionRNN MSE 오차 상한 검증, Z-score 임계치 정상 분류 검증, LLM 설명의 데이터 근거 일치성(할루시네이션 방지) 검증, 윤리 제약조건 준수 여부(의료 최종 진단 문구 포함 금지)를 검증하는 테스트 시나리오를 설계하라.

자동화 코드: tests/test_harness.py 하위에 실제 단위 테스트 코드를 작성하라.

상태 관리 의무: 테스트 실패 사유 및 발견된 결함을 상태 관리에 동기화하라.
